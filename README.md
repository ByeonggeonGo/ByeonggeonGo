## 🌍 Projects by Domain
Here is a list of my key projects grouped by domain.

### 1. GIS & Environmental Science
- [global_temperature_trend](https://github.com/ByeonggeonGo/global_temperature_trend) : Data analysis and visualization analyzing global temperature trends.
- [GeoVision-AI](https://github.com/ByeonggeonGo/GeoVision-AI) : Geospatial imagery and environmental image analysis solutions powered by AI.
- [geo_sdm](https://github.com/ByeonggeonGo/geo_sdm) : Spatial analysis library for Species Distribution Modeling (SDM).
- [CmaqProject](https://github.com/ByeonggeonGo/CmaqProject) : Air quality modeling and analysis project utilizing CMAQ (Community Multiscale Air Quality) model.
- [stream_chem_deg_estimator](https://github.com/ByeonggeonGo/stream_chem_deg_estimator) : Tool for predicting degradation and attenuation of water quality chemical substances in streams.

### 2. Data Analysis & Automation
- [KOSIS_Data_Master](https://github.com/ByeonggeonGo/KOSIS_Data_Master) : A master utility interfacing with KOSIS (Korean Statistical Information Service) OpenAPI to retrieve and download massive statistical data.
- [Langchange-based-EIA-automation](https://github.com/ByeonggeonGo/Langchange-based-EIA-automation) : LLM-based platform automating the reporting and review process for Environmental Impact Assessments (EIA).
- [data_merge_geo_service](https://github.com/ByeonggeonGo/data_merge_geo_service) : Integration and cleaning service for geospatial datasets of various formats.

### 3. AI & Reinforcement Learning
- [multi_agent_awac](https://github.com/ByeonggeonGo/multi_agent_awac) : Research project designing and tuning AWAC (Advantage Weighted Actor-Critic) RL algorithm in multi-agent cooperative environments.
- [dart_rl_up](https://github.com/ByeonggeonGo/dart_rl_up) : A practice project setting up reinforcement learning environments with DART physical simulation.

### 4. Cross-Platform App & Game Development
- [game1](https://github.com/ByeonggeonGo/game1) : 2D game development project using C# and MonoGame to practice core game loop mechanics and graphics rendering.
- [hvac_app](https://github.com/ByeonggeonGo/hvac_app) : Cross-platform utility application for HVAC simulation and data management.
